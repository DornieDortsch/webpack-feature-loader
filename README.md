# webpack-feature-loader
Use webpack to bundle and dynamically load js features on demand.

## Install
```bash
npm install
```

## Start
```bash
npm start
```
Open Browser http://localhost:5000/ and watch console

## Development
```bash
npm run dev
```
Open Browser http://localhost:8080/ and watch console
