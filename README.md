# webpack-feature-loader
Sample project to declarative load js features (modules) with webpack.

## Install
```bash
npm install
```

## Start
```bash
npm start
```
Open Browser http://localhost:5000/ and watch console

## Development
```bash
npm run dev
```
Open Browser http://localhost:5000/ and watch console

or run to watch all tests

```bash
npm run tdd
```

## Test
```bash
npm test
```

## Build
```bash
npm build
```
